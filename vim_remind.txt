# plug 快捷鍵
Shift + e	: nerdtree
Shift + r	: syntastic
Shift + s	: src explore
Shift + t	: tagbar
Shift + u	: undo tree

\\w		: easy-motion
\c<Space>	: commenter
ga		: easy-align

Ctrl + n	: multiple-cursor select

# 控制
qqq		: q
Shift + n	: 行號
Ctrl + y	: file 間複製
Ctrl + p	: file 間貼上
Tab		: indent
Shift + Tab	: unindent	# shift + tab 會被xhsell解析掉，而 vim 下 Ctrl+Tab = Tab
shift + O	: 下一行插入空白行


# 移動於 window 間
Ctrl-w-w 	:move between windows
w+ 上下左右	: Ctrl-w + 上下左右
ww		: Ctrl-w + w

# 移動於 tab 間
Ctrl+PageUp/Down:move between tabs 
gt/gT		:move between tabs

# 移動游標
zz 		: 將當前行移至螢幕中間
H		: cursor移至螢幕頂端
L		: cursor移至螢幕底端
M		: cursor移至螢幕中間
{		: 移至段落開頭
}		: 移至段落結尾
$		: 移至行首
0		: 移至行末
h		: cursor移至螢幕頂端
l		: cursor移至螢幕底端
m		: cursor移至螢幕中間
space		: 下移 10 行
backspace	: 上移 10 行
