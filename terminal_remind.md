# terminal short cut
short cut       | function
----------------|:--------
ctrl + u        | 從游標處向前刪除指令串
ctrl + k        | 從游標處向後刪除指令串
ctrl + a        | 移動游標到行首
ctrl + e        | 移動游標到行尾
ctrl + w        | 刪除一個word
ctrl + y        | paste the words deleted by ctrl + u/k/w
ctrl + p        | view previous command
ctrl + r        | 在歷史裡搜尋指令
sudo !!         | 用 sudo 來執行上個指令
alt + .         | 上個命令後面的參數
ctrl + c        | 中斷執行
ctrl + d        | 中斷輸入
ctrl + z        | 暫停執行
